# ⏳ 1-Minute Countdown Timer

A simple 1-minute countdown timer built with HTML, CSS, and JavaScript.

## 🚀 How It Works

- Click the **Start** button to begin the 1-minute countdown.
- The timer counts down in real time and stops at 0.
